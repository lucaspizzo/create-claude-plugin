## 1.0.0 (2026-03-29)

### Features

- Interactive CLI with plugin name, description, author, and component selection
- Scaffolds Skills, Agents, Hooks, MCP Servers, and LSP Servers
- `--yes` flag for non-interactive mode
- `--no-git` flag to skip git initialization
- Generates `.gitignore`, `LICENSE`, and `README.md` in plugins
- Auto-initializes git repository
- Colored terminal output with `NO_COLOR` support
- Input validation and error handling with cleanup on failure
